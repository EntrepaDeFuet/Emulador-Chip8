# Emulador-Chip8
Emulador de chip8 programat en C per mitjà de la llibreria SDL2.

Per tal de programar-lo s'ha seguit part del tutorial del canal de youtube:
Queso Fuego.

Els roms, s'han extret d'altres repositoris públics de github amb l'objectiu
de comprovar el correcte funcionament del emulador.

Aquest és un projecte sense ànim de lucre, amb l'únic objectiu d'aprendre.
